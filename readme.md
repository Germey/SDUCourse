## 山东大学刷课系统

### 系统简介

本系统为山东大学选课刷课专用，配置好学号、密码、课程号、课序号即可完成多线程无限刷课，直至选中该课程为止。

本系统基于Python 2.7编写，支持多线程刷课，掉线自动登录，登录失败重试，选中课程即停等功能。

线程数目、登录失败重试次数、超时时间、学号、密码、课程号、课序号均可自定义配置。

### 编写语言

Python 2.7
